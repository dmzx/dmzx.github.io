### Jekyll

Create new draft
```shell
bundle exec rake new:draft 'Title'
```

Create new post
```shell
bundle exec rake new:post 'Title'
```

Create new page
```shell
bundle exec rake new:page 'Title'
```

### Satis

Update Satis
```shell
composer update --prefer-dist
```

Build repository
```shell
vendor/bin/satis build -nv
```
